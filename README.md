# Harf Lab SaaS Starter

Starter kit for a real Harf Lab SaaS connected to Supabase.

## What's included
- Database schema SQL
- Row Level Security policies
- React + Vite frontend
- Supabase Auth
- Customer pages
- Admin dashboard pages
- Order creation and tracking starter logic

## Run locally
1. Copy `.env.example` to `.env`
2. Add your Supabase URL and anon key
3. Run:
   npm install
   npm run dev

## Deploy
- Push to GitHub
- Import into Vercel
- Add environment variables in Vercel

## Important
This is a starter architecture and codebase, not a fully finished production app.
